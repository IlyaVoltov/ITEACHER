Объяснение решения задачи:

Ввод данных
Количество учеников:


students_count = int(input("Введите количество учеников: "))
Запрашиваем у пользователя число, указывающее количество учеников.

Словарь для хранения данных:


students = {}
Создаём пустой словарь, где:

Ключи — имена учеников.
Значения — вложенные словари с предметами и оценками.
Цикл для ввода данных о каждом ученике:


for _ in range(students_count):
Цикл повторяется столько раз, сколько указано в students_count.

Ввод имени и оценок:


name = input("Имя ученика: ")
subjects_scores = input("Введите предметы и оценки через запятую (например, Математика:5, Физика:4): ")
Для каждого ученика запрашиваем его имя и строки с предметами и оценками.

Разбиение строки с предметами и оценками:


subjects_scores_dict = {}
for item in subjects_scores.split(', '):
    subject, score = item.split(':')
    subjects_scores_dict[subject] = int(score)
subjects_scores.split(', ') разбивает строку по запятым и пробелам, чтобы получить отдельные предметы и оценки.
Затем каждый элемент снова разбивается по символу ':' для выделения предмета и оценки.
Создаётся словарь subjects_scores_dict, который добавляется в students.
Сохранение данных об ученике:


students[name] = subjects_scores_dict
Обработка и вывод результатов
Перебор учеников и подсчёт средней оценки:


for name in students:
    scores = students[name].values()
    average_score = sum(scores) / len(scores)
    print(f"{name}: средняя оценка {average_score:.1f}")
students[name].values() получает все оценки.
sum(scores) / len(scores) вычисляет среднюю оценку.
Результат выводится с использованием f-строки.
Поиск лучшего ученика:


if average_score > highest_average:
    highest_average = average_score
    best_student = name
Сравниваем среднюю оценку каждого ученика с текущей самой высокой и обновляем переменные, если новая выше.

Вывод лучшего ученика:


print(f"Лучший ученик: {best_student} (средняя оценка {highest_average:.1f})")
Это решение иллюстрирует работу с вложенными словарями, строками, циклами и условными конструкциями для решения практической задачи.
